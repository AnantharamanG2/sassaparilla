Sassaparilla

Download or Fork Me

Great Typography and Responsive Web Design, right off the bat

Sassaparilla is an open source, fast way to start your responsive web projects that harnesses the power of SASS and Compass

A core focus on better typography

Nail your vertical rhythm with some help from compass’ awesome typography module. You’ll never need to spend hours on calculations again, giving you more time to focus on creating better typography and overall rhythm to your designs.

But also…

Work in px, compile in ems

Most of us don’t like hardcore maths. Wouldn’t it just be easier if we could work in pixels and let something else do the work?

No more than you need

We’ve tried not to make assumtions as to what your project may need. Sassaparilla is just about getting you started.

Work with css variables

If you like using variables in your css for things like colour you can using sass, and Sassaparilla keeps them all in one place for safe keeping.

Made by makers like you

We use this thing, and have been iterating and improving it for some time. We find it invaluable here at fffunction, and we hope you will too.

Setup is simple too

If you’ve used compass and sass before, or if you’ve never tried them, getting started is really easy. We’ve tried to make it as painless as possible, using as little set-up time as required.

Don’t believe us? Then have a nose at the read me.
